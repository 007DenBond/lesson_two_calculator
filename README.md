# Семейный бюджет

Небольшой веб-проект для учета личных/семейных финансов: доходы, расходы, аналитика по категориям и цели накоплений.

## Что уже есть

- приветственный лендинг-блок на главной странице;
- удобный ввод доходов и расходов;
- аналитика: прогресс-бар, категории и круговая диаграмма;
- цели накоплений с пополнением и процентом выполнения;
- экспорт данных в JSON;
- светлая и темная тема;
- сохранение данных в `localStorage` по месяцам.

## Стек

- `HTML`
- `CSS`
- `JavaScript` (vanilla)

## Быстрый старт

1. Откройте файл `Calculator.html` в браузере.
2. Добавьте первый доход и расход.
3. Следите за балансом и советами.

## Структура проекта

- `Calculator.html` - основной файл приложения и лендинга.
